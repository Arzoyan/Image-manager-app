# Image Manager App

## Description

Using this project you can Adding, editing and deleting of posts with uploading images

## Installation

```bash
$ git clone https://github.com/Arzoyan/Image-manager-app.git
```

navigate to root directory

```bash
$ cd image-manager-app

```

Install project

```bash
$ npm install
```

## Running the app

```bash
$ npm run dev
```
